# Changelog

## [0.1.0] - 2026-04-20 
### Додано
- Початкова структура проєкту та налаштування файлу `.gitignore`.
- Модуль завантаження даних для набору даних Гарячої лінії Львова за 2024 рік.
- Скрипт аналізу якості даних (виправлення помилок у координатах та перетворення форматів дат).
- Модуль дослідження даних (кластеризація за допомогою K-Means та аналіз ефективності виконання заявок).
- Модуль просунутої візуалізації (геопросторові графіки, аналіз категорій та діаграми продуктивності).

# Аналіз звернень на Гарячу лінію Львівської МТГ за 2024 рік

## Мета проєкту: 
Провести аналіз просторового розподілу та тематичної структури звернень мешканців Львівської громади на гарячу лінію у 2024 році, виявити проблемні райони та оцінити ефективність обробки заявок.

## Джерело даних
Дані отримані з порталу відкритих даних: Дані про надходження звернень на гарячі лінії (2024).
Використовується файл "glm_all_2024_portal.csv" — звернення за увесь 2024 рік з повною структурою полів.

## Завдання
1. На основі колонок "latitude" та "longitude" кластеризувати звернення за географічним розташуванням та виявити райони міста ("district") з найвищою концентрацією проблем.
2. Проаналізувати колонку "category" для визначення топ-10 тематичних категорій звернень та їхньої частки від загального обсягу, з розбивкою за типом ("type") та джерелом надходження ("appealSource").
3. На основі полів "registrationDate", "executionDate" та "executionStatus" обчислити середній час виконання звернень і порівняти ефективність різних виконавців ("executantName") по категоріях.
